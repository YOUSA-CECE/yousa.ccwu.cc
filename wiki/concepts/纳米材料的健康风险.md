
# 纳米材料的人体健康风险

## 1. 人体暴露途径

### 1.1 呼吸摄入（首要途径）
- 纳米颗粒可穿透上呼吸道屏障，深入沉积于肺泡区
- 难以被呼吸道黏液-纤毛系统清除，大幅延长肺部滞留时间
- **<20 nm 的超细颗粒（UFPs）** 可穿过肺泡上皮进入血液循环
- 分布至肝、脾、脑等全身器官
- **职业暴露**：纳米材料生产车间浓度远高于普通环境，是主要健康威胁

### 1.2 消化道摄入
- 来源：受污染的食物和饮用水、添加纳米材料的食品/保健品/个人护理产品
- 50 nm 聚苯乙烯纳米颗粒能跨越肠道屏障进入小鼠循环系统
- 纳米材料可改变肠道菌群组成与代谢活性，通过"肠-轴"通讯影响免疫和代谢

### 1.3 皮肤接触
- TiO₂、ZnO 等纳米颗粒可经毛囊、皮脂腺侵入
- 皮肤受损时渗透率显著提升
- 可能诱发局部炎症反应或经淋巴循环引发系统性免疫激活

### 1.4 静脉注射（纳米医药）
- 纳米药物递送的核心途径
- 可能引发过敏反应、干扰凝血功能
- 易在肝、脾等网状内皮系统器官蓄积

---

## 2. 跨膜运输

纳米材料进入细胞的主要过程：细胞膜表面吸附 → 嵌入磷脂双层 → 非特异性物理穿透 → 细胞内吞作用

**影响因素（以内吞作用为例）：**

| 因素 | 影响 |
|------|------|
| **尺寸** | 20-50 nm 最易被细胞摄取；过小易被排出，过大难以摄取 |
| **几何形状** | 球形 → 网格蛋白介导内吞（CME）；棒状/管状 → 巨胞饮/小窝蛋白介导 |
| **表面电荷** | 阳离子颗粒摄取效率最高；过高正电性破坏细胞膜 |
| **表面修饰** | PEG 增加亲水性与循环时间；靶向配体提升靶向摄取 |

---

## 3. 生物降解

| 机制 | 描述 |
|------|------|
| **酶促氧化降解** | 髓过氧化物酶（MPO）等氧化切割碳纳米管（CNTs）/氧化石墨烯（GO）的 C-C 键 |
| **酶促还原降解** | 缺氧环境下特定还原酶将金属氧化物还原为易溶形态 |

**影响因素：** 化学组成、尺寸、表面官能团、结晶度。带羧基/羟基亲水基团的 CNTs 更易被酶识别降解。

**安全性启示：** 可酶解的纳米材料通常表现出更优的生物相容性，减少慢性炎症和器官损伤风险。

## 4. 毒性效应

### 呼吸系统
| 效应 | 机制 |
|------|------|
| 急性炎症反应 | 肺泡巨噬细胞吞噬后激活炎症信号，释放促炎细胞因子 |
| 氧化应激损伤 | 活性位点/催化作用产生 ROS，脂质过氧化和 DNA 损伤 |
| 慢性肺纤维化 | 长期暴露（如碳纳米管）导致不可逆纤维化病变 |
| 肺功能不可逆下降 | 肺顺应性降低，肺活量下降 |

### 基因和遗传毒性
1. **直接损伤**：纳米颗粒边缘/表面直接与 DNA 物理接触，造成链断裂
2. **间接损伤**：ROS 攻击 DNA 碱基（最主要机制）
3. **表观遗传效应**：影响 DNA 甲基化、组蛋白修饰

### 其他器官毒性
| 器官 | 毒性表现 |
|------|---------|
| **肝脏** | 主要蓄积器官，氧化应激、肝细胞坏死、慢性肝纤维化 |
| **肾脏** | 肾小球滤过后滞留于肾小管，上皮细胞损伤 |
| **皮肤** | 接触性皮炎，增强对过敏原的免疫应答 |
| **心血管** | 损伤血管内皮，破坏屏障完整性，加速动脉粥样硬化 |

---

## 5. 毒性作用机制

### 5.1 细胞器损伤

**溶酶体膜破坏：**
- 阳离子纳米颗粒与溶酶体膜阴离子磷脂结合
- 膜破裂 → 酸性水解酶释放 → 细胞自溶与坏死

**线粒体功能紊乱：**
- 破坏跨膜电位，抑制 ATP 合成
- 诱导线粒体呼吸链产生过量 ROS
- 激活 Caspase 蛋白酶家族 → 细胞凋亡

### 5.2 活性氧（ROS）相关毒理（核心机制）

| 来源 | 机制 |
|------|------|
| **表面催化** | 不饱和键/高活性原子催化氧气或水产生 O₂⁻、H₂O₂ |
| **金属离子释放** | Fe²⁺、Cu²⁺ 通过 Fenton 反应产生 ·OH |
| **破坏抗氧化平衡** | 消耗 GSH、抑制 SOD 活性，打破氧化-抗氧化平衡 |

过量 ROS 无差别攻击脂质、蛋白质和 DNA，导致细胞膜脂质过氧化、酶活性丧失及 DNA 突变。
